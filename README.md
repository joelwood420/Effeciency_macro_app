# Effeciency_macro_app

A web application for tracking daily macronutrient intake and managing fitness goals. Designed for fitness enthusiasts who want to log their meals and monitor protein, carbs, and fat intake with precision.

## Features

- **Breakfast Macro Logging**: Launch screen to quickly log breakfast macros
- **Individual Macro Input Fields**: Separate input fields for protein, carbohydrates, and fats
- **Skip Entry Functionality**: Option to skip meals without losing tracking continuity
- **Weekly Totals & Averages**: Automatically calculate weekly macro totals and daily averages
- **Responsive Design**: Works on desktop and mobile devices
- **Real-time Calculations**: Instant feedback on macro targets and remaining allowances

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript
- **Architecture**: Client-side web application
- **Storage**: Browser local storage for persistent data

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/joelwood420/Effeciency_macro_app.git
```

2. Navigate to the project directory:
```bash
cd Effeciency_macro_app
```

3. Open `index.html` in your web browser

### Usage

1. Launch the application by opening `index.html`
2. Input your breakfast macros in the dedicated fields
3. Use the skip entry option if you don't want to log a specific meal
4. View your weekly totals and averages in the summary section
5. Data persists locally in your browser
